# Calendario contenuti social

Web app statica per pianificare contenuti social con login Google/Firebase e dati condivisi su Cloud Firestore.

## Pubblicazione con GitHub Pages

1. Crea un repository GitHub.
2. Carica tutti i file di questa cartella nella root del repository.
3. Vai su `Settings > Pages`.
4. In `Build and deployment`, scegli `Deploy from a branch`.
5. Seleziona branch `main` e folder `/root`.
6. Salva.

Dopo la pubblicazione, aggiungi il dominio GitHub Pages in Firebase:

`Authentication > Settings > Authorized domains`

Esempio:

`tuo-username.github.io`
